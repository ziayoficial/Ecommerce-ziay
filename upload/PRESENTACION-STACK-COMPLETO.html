Stack Overview

Cómo implementamos los 4 ejes enterprise — capa por capa

El stack técnico de ZIAY implementa los 4 ejes enterprise (Crecimiento medible, Eficiencia operativa, Gobernanza, Integración) a través de 3 capas de arquitectura: Revenue Layer, Operations Layer y Governance Layer.

Helper HMAC (src/lib/middleware/hmac.ts)

import crypto from 'crypto';

export function verifyMetaSignature(rawBody: string, signature: string, appSecret: string): boolean {
  if (!signature || !appSecret) return false;
  const expected = 'sha256=' + crypto.createHmac('sha256', appSecret).update(rawBody).digest('hex');
  return crypto.timingSafeEqual(Buffer.from(signature), Buffer.from(expected));
}
